## Django проект для управления уроками и студентами с использованием Celery и Redis.
http://localhost:8000
## Приложение о
1. По умолчанию имеет трех студентов, которые уже записаны на один урок.
2. При создании "своего" урока все три студента автоматически записываются на него и получают уведомления.
3. Есть два типа уведомлений:
- уведомление по расписанию.
  - селери-бит ищет ближайшие уроки и отправляет уведомление об их скором начале.
- уведомление по регистрации.
  - после создания урока селери отправляет уведомление о том. что студент на него зарегистрирован.


## Запуск приложения.
### Команды для настройки PyCharm(разработка):
- manage.py runserver;
- celery -A django_lesson worker --loglevel=info;
- celery -A django_lesson beat --loglevel=info;(по расписанию).
- python manage.py migrate - миграция;
-  python manage.py makemigrations - создание миграции;
