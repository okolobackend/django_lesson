## Django проект для управления уроками и студентами и учителями(в будущем) с использованием Celery, Redis и Django Signals.

## Проверка
### Celery
1. По умолчанию есть три студента, которые уже записаны на один урок.
2. Каждые десять минут селери-бит ищет ближайшие уроки и планирует напоминание о старте занятия(селери).
### Django signals
1. По созданию нового урока(через форму на странице) будет простая запись в лог о том, что в систему добавлен новый урок и о то, что туда зарегистрированы имеющиеся студенты.
2. Также, но уже через селери, отправляем студентам уведомление об уроке.


## Запуск приложения через докер.
Требования:
- Docker(проверено с Docker version 29.0.2)
- Docker Compose(проверено с Docker Compose version v2.40.3)
Инструкция:
1. Склонируйте репозиторий
2. Запустите проект:
   ```bash
   docker compose up
3. Откройте в браузере http://localhost:8000
